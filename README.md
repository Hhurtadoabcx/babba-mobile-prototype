# Babba Mobile Sensors Prototype

Aplicación móvil en React Native/Expo para capturar datos del giroscopio del dispositivo como parte del sistema Babba de monitoreo IoT.

## Tecnologías
- React Native
- Expo
- Expo Sensors
- JavaScript

## Funcionalidad actual
- Lectura del giroscopio del dispositivo.
- Visualización de coordenadas X, Y, Z.
- Base para integración con backend Babba.
